# 公開授權聲明

Static CID Governance 及其相關 Offline Editor 參考元件，由 Stephen Hsu（許家誠）公開釋出，作為非營利、教育、公共利益、數位典藏與靜態發布場景可參考使用的開放實作。

本公開授權的目的，是讓非營利組織能夠在明確授權條款下，合法研究、採用、修改與改良此模型。本案並非以專利壟斷作為推廣方式，而是提供一套可被檢查、可被學習、可被延伸的治理基礎，用於內容完整性、離線編輯、靜態發布、引用一致性與 public/private 資料邊界保護。

本專案採用分層授權模式：

- 核心治理工具採用 GPL-3.0-or-later。
- 公開範例、schema、policy、template 與 sample static-site 元件採用 MIT。
- 文件、圖表、報告與教學材料採用 CC BY-SA 4.0。

因此，非營利組織可以依各檔案與目錄所標示之授權條款使用本專案，同時保留作者署名，並遵守各授權條款對開放性、散布與改作的要求。
