# Open Reference Release Notes v1.0.0

This release cuts out a neutral open reference implementation from a broader private working environment.

## Included

- Static CID Governance model documents.
- FHIR/CID concept mapping for non-medical settings.
- Mermaid diagrams.
- Minimal public static-site example.
- Defensive Python tools for CID generation, public projection, reference verification, and public inventory guarding.
- Layered license structure.

## Excluded

- Organization-specific public website output.
- Press and profile media.
- Private registries and production manifests.
- Credential, ACL, account recovery, or deployment tooling.
- Any content not solely owned or intentionally licensed by the author.
