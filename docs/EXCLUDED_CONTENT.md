# Excluded Content

The public open reference release deliberately excludes content and operational materials that are not necessary to understand the model.

Excluded materials include:

- organization-specific website content;
- third-party or press media;
- private registries;
- credentials, tokens, certificates, and secrets;
- internal account policies;
- account recovery workflows;
- production deployment settings;
- private governance logs;
- production package manifests;
- incident-specific or organization-specific security records.

This repository is intended to publish a neutral model, not an operational copy of any specific production system.
